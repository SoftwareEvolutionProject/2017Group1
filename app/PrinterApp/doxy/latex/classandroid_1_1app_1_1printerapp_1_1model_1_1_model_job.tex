\subsection{Detailed Description}
This class defines a new Printing Job as a Status listener. Basically it\textquotesingle{}s a reference to the current status of the printer. If there is currently an ongoing job, it\textquotesingle{}ll show the printing status, if there\textquotesingle{}s nothing, it won\textquotesingle{}t show anything. \begin{DoxyAuthor}{Author}
